# collegesathi
A simple college carpool web app where students can share rides based on routes and locations.
📌 Features

👤 User Registration & Login

🚗 Create Ride (with start location and via points)

🔍 Search Rides by location / landmark

📍 Route-based matching (start + via locations)

💸 Simple bidding system for ride requests

📅 Date, Day, and Time selection (AM/PM)

🔄 Daily ride update option

🛠️ Tech Stack

Backend: Python (Flask)

Frontend: HTML, CSS, JavaScript

Database: SQLite

⚙️ How It Works

Driver creates a ride with:

Start location

Multiple via points

Time and date

Passenger searches using a location

If location matches:

Start point OR

Any via point

👉 Ride is shown

Passenger sends request with offer price

Driver accepts or rejects
